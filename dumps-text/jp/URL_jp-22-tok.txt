 複数 形 の フェミニズム

     FEMINISMS 」 展 は タイトル が 示す 通り 、 複数 形 の フェミニズム を テーマ に し て いる 。 フェミニズム と いう と 様々 な 歴史 や 理論 、 アクティビズム 、 表象 
     等 を 含む 非常 に 大きな テーマ だ が 、 ここ で は その 総体 を 概観 する と いう より も 、 あくまで も 個別 の 視点 、 つまり アーティスト たち の 作品 を 通し て 、 フェミニズ

     か ？ 　 赤 と いう 強烈 な 色 で 写真 を 覆う こと で 、 作家 は 家族 の イメージ が 孕む 「 違和 感 」 の 根本 へ と 注意 を 促し 、 その 違和 感 が 変質 し たり 消え 去る よう な 未来 を

     ション を あえて つくり だし 、 インター プレイ な 波上 を サーフィン する 。 遊戯 的 で 政治 的 な この 作品 を 見 て いる と 、 粘土 と とも に 規範 や 固定 観念 も もみ ほぐれ さ
